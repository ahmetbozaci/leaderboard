### Setup
To run code locally, open your terminal and follow these steps:

- `git clone (git@github.com:ahmetbozaci/leaderboard.git)`
- `cd leaderboard` 
- `npm install`
- `npm start` 
  
*After `npm start` webpack will automatically open your browser.*
